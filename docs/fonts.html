Install fonts

Download the following font files according to the dictionary language into the koreader/fonts folder.

Pronunciation fonts

  • Gentium Plus: GentiumPlus-Regular.ttf
  • Noto Sans Mono: NotoSansMono-Regular.ttf

Chinese fonts

  • Noto Sans Traditional Chinese:
    • NotoSansTC-Regular.ttf
    • NotoSansTC-Bold.ttf
  • Noto Sans Simplified Chinese:
    • NotoSansSC-Regular.ttf
    • NotoSansSC-Bold.ttf
